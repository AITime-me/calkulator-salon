# Месячный бюджет салона красоты

Одностраничное веб-приложение для учёта доходов и расходов салона красоты по месяцам. Работает полностью в браузере, без сервера и внешних библиотек.

## Возможности

- ведение бюджета по месяцам с выбором периода;
- ежедневный учёт операций по конкретной дате;
- автоматическое отнесение операций к месяцу по дате;
- расчёт итогов и чистой прибыли за выбранный месяц;
- история операций с удалением отдельных записей;
- очистка всех операций выбранного месяца;
- расшифровка сумм по категориям доходов и расходов;
- сохранение данных в `localStorage` браузера;
- адаптивный интерфейс для телефона и компьютера.

## Быстрый старт

1. Склонируйте репозиторий или скачайте архив проекта.
2. Откройте файл `index.html` в любом современном браузере.
3. Выберите месяц бюджета и начните добавлять операции.

Дополнительная установка и сборка не требуются.

## Расчёты за выбранный месяц

| Показатель | Формула |
|---|---|
| Общий доход | сумма всех доходов |
| Расходы без ФОТ и аренды | все расходы, кроме «Фонд оплаты труда» и «Аренда» |
| Фонд оплаты труда | сумма по категории «Фонд оплаты труда» |
| Аренда | сумма по категории «Аренда» |
| Итого расходов без аренды | расходы без ФОТ и аренды + ФОТ |
| Итого расходов с арендой | итого без аренды + аренда |
| Чистая прибыль | общий доход − итого расходов с арендой |

## Хранение данных

Все операции сохраняются в браузере под ключом `salon-budget-operations`.

Данные:

- сохраняются автоматически при каждом изменении;
- остаются после перезагрузки страницы;
- привязаны к конкретному браузеру и устройству;
- удаляются при очистке данных сайта в браузере.

## Проверка сценариев

1. Добавьте доход за текущий месяц — проверьте итоги и уведомление «Доход добавлен».
2. Добавьте расход за текущий месяц — поле суммы должно очиститься.
3. Добавьте операцию с датой из другого месяца — месяц бюджета переключится автоматически.
4. Смените месяц вручную — история и итоги обновятся.
5. Удалите одну операцию кнопкой ✕.
6. Очистите выбранный месяц кнопкой «Очистить месяц».
7. Перезагрузите страницу — данные должны сохраниться.
8. Попробуйте отправить форму без даты — появится inline-ошибка.
9. Попробуйте указать сумму `0` или отрицательное значение — появится сообщение об ошибке.
